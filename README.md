# Notebooks Tutorial

Welcome to the Notebooks Tutorial Library of Data Juicer Hub! This folder contains a series of Jupyter notebooks to help you get started with Data Juicer quickly.

## ðŸš€ Quick Start

### Method 1: Codespace (Recommended)

1. **Launch Codespace**
   - Click the `Code` button on this repository and select the `Codespaces` tab
   - Click `Create codespace on notebook` to launch the environment
   - Wait a moment, and you will see the VSCode Web interface

2. **Select and Run Notebook**
   - Find the `notebooks` folder in the left file directory
   - Click on the notebook file you're interested in
   - Click the kernel selector in the top-right corner and select the **`data-juicer-hub`** environment (located in the `.venv` directory)
   - Start running!
